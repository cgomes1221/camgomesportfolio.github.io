Latest Projects

Crosshair Sports - The free mobile application to help athletes get noticed

Crosshair Sports is a free to use mobile application created to help athletes promote themselves and get noticed by coaches. Many sports recruiting services have become expensive and hard to obtain for many families. With sports becoming more comopetitive than ever before, it is important for athletes to be able to receive the exposure needed in order to be seen by coaches at the next level. Some features of this application include: the ability to post pictures and videos, direct messaging that allows players and coaches to communicate with each other, a news feed that allows athletes and coaches to share important announcements with one another, and the ability to search for users. This application is now available to download on the AppStore. This mobile application was created using Dart and Flutter.

Steelo (UI) E-Commerce Mobile Application with a Twist

Steelo is a UI mobile application that combines aspects of dating apps such as Tinder, with ecommerce applications such as Poshmark. The first screen shown to users allows them to swipe right or left on clothing that they may or may not be interseted in buying. Swiping right would add the product to the users cart while swiping left would show that they do not have interest in the product. Other available pages include an explore page, that shows popular styles that other users are wearing and a page that shows the user's closet which would show items that the user would want to sell to others. This application was created with Dart and Flutter.
